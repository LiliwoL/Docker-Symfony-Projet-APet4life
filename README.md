# Commandes

## Kill

`bin/kill`

## Start Containers

`bin/start`

## Stop container

`bin/stop`

## Shell

`bin/shell`

***

# Configuration

Dans le fichier .env, renseignez les champs nécessaires

# Installation

Après avoir démarré les containers avec `bin/start`

Faites un `bin/shell` puis installez les **dépendances composer**

```bash
composer install
```
